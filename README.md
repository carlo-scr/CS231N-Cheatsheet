# CS231N Midterm Cheatsheet

> A compact 2-page reference sheet for the **Stanford CS231N** (Deep Learning for Computer Vision) midterm.

## Contents

| Topic | Key Concepts |
|---|---|
| Image Classification | kNN, linear classifiers, SVM, Softmax |
| Optimization | SGD, momentum, learning rate schedules |
| Neural Networks | Backprop, activations, weight initialization |
| CNNs | Conv layers, pooling, receptive fields, architectures |
| Training | Batch normalization, dropout, regularization |

## Building

Requires a LaTeX distribution with `pdflatex`. Compile with:

```bash
pdflatex main.tex
```

This produces `main.pdf` — 2 pages, A4 landscape.
